# Data-Categorization-Automation

Project Overview

This project automates the categorization and normalization of organizational data stored in Excel files.

Using Python, multiple organization names were grouped into standardized main actors to improve data consistency and reporting accuracy.

The automation significantly reduced manual categorization time from hours to seconds.

Main Features
- Automated categorization process
- Data normalization
- Actor standardization
- Record counting by category
- Excel data processing
  
Technologies Used
- Python
- Pandas
- Excel Automation
  
Workflow
1. Read Excel database
2. Identify organization names
3. Map organizations into main actor categories
4. Normalize dataset
5. Generate categorized outputs and record counts
   
Important Note

All data included in this repository has been anonymized and modified for portfolio purposes only.

Additional Impact

The categorization logic developed in this project was later adapted and reused across multiple operational scenarios within the organization, supporting different data classification and normalization needs.

This improved consistency, reduced repetitive manual work, and standardized reporting processes across the area.
